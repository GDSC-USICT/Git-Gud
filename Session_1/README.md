Git commands covered in first session  **(date : 5th Sept 2021)**

1. Working on separate branch ( not master/main )

 ```bash
$ git branch 
$ git checkout -b
```

2. rewriting git history 

 ```bash
$ git rebase -i <till where>
$ git commit --amend
```

3. Undoing changes ( and difference between them )

```bash
$ git revert
$ git reset
$ git checkout
```


4. viewing doc of a command

```bash
$ git help <command_name>
```


5. Git reflog 

```bash
$ git reflog
```
